English keyboard keys / Teclas en el teclado inglés:

One key on English keyboard plus (+) another key if necessary = (converts to) key desired
Una tecla en el teclado inglés plus (+) otra tecla si es necesario  = convierte a la clave deseada

Punctuation / Puntuación
? + ? = ¿   inverted question mark / signo de interrogación invertido
! + ! = ¡   c U+00A1   inverted exclamation mark / signo de exclamación invertido
< = ‘    c U+2018   right pointing single quote mark / marca de comillas simples que apunta hacia la derecha
> = ’    c U+2019   left pointing single quote mark / marca de comillas simples que apunta hacia la izquierda
 + < > “   c U+201C   right pointing double quote marks / marca de comillas dobles que apunta hacia la derecha
’ + = > ”   c U+201D   left pointing double quote marks / marca de comillas dobles que apunta hacia la izquierda
/ + < = «   c U+00AB   left pointing double angle quotation mark /  marca de doble ángulo que apunta hacia la izquierda
/ + > = »   c U+00BB   right pointing double angle quotation mark / marca de doble ángulo hacia la derecha
\ + ] = ]   c U+005C   right square bracket / corchete derecho
{ + { = }   c U+007B   right curly bracket / corchete curvo derecho
| + < = <   c U+007C   left pointing wedge / ángulo que apunta a la izquierda
| + > = >   c U+007C + 003E  right pointing wedge / ángulo que apunta a la derecha
= + = = —   c U+2014    M—dash  / raya larga 
- + - = –   c U+2013    N-dash / raya más larga que el guión pero menos corto que la raya larga
; + ; = ˈ   c U+02C8   small, glottal stop / oclusiva glotal (pequeña)
Special Upper case Vowel / Caracter especial mayúscula
 } = Ʉ      c U+0244
Special Lower case vowel / Caracter especial minúscula
] = ʉ      c U+0289

Acute Accented Vowels / Vocales mayúsculas acentuadas
; + A = Á     c U+00C1
; + E = É     c U+00C9
";" + I = Í     c U+00CD
";" + O = Ó     c U+00D3
";" + "U" = "Ú"     c U+00DA   
";" + } = Ʉ́     c U+0244 + 0301

Acute Accented Lowercase Vowels / Vocales minísculas acentuadas
";" + a = á     c U+00E1
";" + e = é     c U+00E9
";" + i = í     c U+00ED
";" + o = ó     c U+00F3 
";" + u = ú     c U+00FA
";" + ] = ʉ́     c U+0289 + 0301

Nasalized Uppercase vowels  / Vocales mayúsculas nasalizadas
[ + A = Ã     c U+00C3
[ + E = Ẽ     c U+1EBC
[ + I = Ĩ     c U+0128
[ + O = Õ     c U+00D5
[ + U = Ũ     c U+0168
[ + } = Ʉ̃     c U+0244 Ʉ + 0303 combining tilde

Nasalized Lower case Vowels / Vocales minúsculas nasalizadas
[ + a = ã    c U+00E3
[ + e = ẽ    c U+1EBD
[ + i = ĩ    c U+0129
[ + o = õ    c U+00F5
[ + u = ũ    c U+0169 (nasalized u)
[ + ] = ʉ̃    c U+0289 followed by U+030 combining tilde

Spanish eñe / La eñe
{ + N = Ñ   c Upper case
[ + n = ñ   c Lower case U+00F1

Spanish diaeresis / La U con diaresis
: + U = Ü    c Upper case U+00DC
: + u = ü    c Lower case U+00FC
